## WEEKLY TASKS (Every Week)

### Google Business Profile
- [ ] Post 1 update with photo (Monday recommended)
- [ ] Respond to any new reviews within 24 hours
- [ ] Check for and answer any new Q&A questions
- [ ] Upload 2-3 new photos from recent work

### Review Generation
- [ ] Send review request to 1-2 recent clients
- [ ] Follow up on pending review requests (3-day old)

### Social Media (Supports SEO)
- [ ] Post 2-3 times on Instagram/Facebook
- [ ] Share portfolio work with location tags
- [ ] Engage with local business posts

## MONTHLY TASKS (First Week of Month)

### Content Creation
- [ ] Publish 1 blog post (500-1000 words, local keywords)
- [ ] Update any outdated service pages
- [ ] Add new portfolio items with descriptions

### Google Business Profile
- [ ] Add 10+ new photos
- [ ] Update any services or offerings
- [ ] Post 4 GBP updates (weekly)
- [ ] Review and update business description if needed

### Citations & Directories
- [ ] Check top 10 directories for accuracy
- [ ] Update any changed information
- [ ] Look for new directory opportunities

### Analytics Check
- [ ] Review Google Search Console
  - Check for crawl errors
  - Review top queries
  - Check impressions/clicks trend
- [ ] Review Google Business Profile Insights
  - Total views
  - Search queries used
  - Actions taken (calls, website clicks, directions)

## QUARTERLY TASKS (Every 3 Months)

### Comprehensive Audit
- [ ] Test all pages in Google Rich Results Test
- [ ] Verify schema markup is working
- [ ] Check all internal links work
- [ ] Review page load speed (Google PageSpeed Insights)
- [ ] Check mobile friendliness

### Competitor Analysis
- [ ] Search main keywords, note who ranks
- [ ] Review competitor GBP profiles for new ideas
- [ ] Check competitor reviews/ratings
- [ ] Identify content gaps to fill

### Citation Audit
- [ ] Verify NAP consistency across all directories
- [ ] Look for new citation opportunities
- [ ] Remove or update any duplicate listings

### Content Refresh
- [ ] Update pricing if changed
- [ ] Refresh outdated statistics or information
- [ ] Add new testimonials to pages
- [ ] Update portfolio with recent work
